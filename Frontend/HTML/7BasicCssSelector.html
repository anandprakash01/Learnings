CSS stands for Cascading Style Sheet

In Internal or External CSS which property is written later that will be taking the precedence

A CSS selector selects the HTML element(s) you want to style.

CSS selectors are used to target the html elements in markup

CSS Selectors makes it easy for us to easily target single/multiple HTML elements in the markup

id is unique and class can be multiple

Types of basic CSS selectors

  • CSS element selector
  • CSS id selector
  • CSS class selector
  • Css grouping selector
  1. The element selector selects HTML elements based on the element name.
  2. The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element
  3. The class selector selects HTML elements with a specific class attribute.
  4. The grouping selector selects all the HTML elements with the same style definitions.
  5. The universal selector (*) selects all HTML elements on the page.

Inline CSS is given precedence over Internal & external CSS
